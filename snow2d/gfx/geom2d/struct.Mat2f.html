    pub m22: f32,

Expand description

Column-major 2x3 matrix, which represents 3x3 matrix

 m[0] m[1]  ...  |  scale_x  sin       0
 m[2] m[3]  ...  |  cos      scale_y   0
 m[4] m[5]  ...  |  trans_x  trans_y   1
