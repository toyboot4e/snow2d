Crate snow2d::utils::pool[−]

Expand description

Pool with reference-counted items

Items in the Pool will be reference-counted with strong Handles. When no Handle is referring to an item, it can be removed on synchronization, or you can handle it manually.

Note that the pool does NOT drop unreferenced items until it’s synced. Also it’s single-thread only, for no particular reason.
