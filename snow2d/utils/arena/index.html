Crate snow2d::utils::arena[−]

Expand description

Extensible generational arena for various uses. Example

Goals: Tiny code and real use. Non-goals: Super fast performance and no UB.

Features

Flexibilities:

  • Builtin support for distinct arena types (second type parameter of Arena<T, D, G>).
  • Customizable generation type (third type parameter of Arena<T, D, G>).

Unsafe goodies:

  • (UB) Mutable iterator of entries (Arena::bindings), rather than slot/index-based iteration.
  • (UB) Virtual Arena<RefCell<T>> mode: (Arena::cell).

Restrictions:

  • Single-threaded
